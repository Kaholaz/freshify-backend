# Freshify backend
This is the backend for freshify.no. The backend can be accessed on api.freshify.no. It can also be run locally.

## Dependencies
- Docker v23.0.2
- Apache Maven v3.6.3
- OpenJDK Runtime Environment v17.0.6

## How to run
### Linux
```
cd api
docker pull mysql
./start_application.sh
```
